# Formulir Pendaftaran IMPHNEN

## 📌 Deskripsi Proyek

Project ini merupakan tugas TM3 pada mata kuliah Workshop Desain UI yang berfokus pada penerapan JavaScript dasar dalam pembuatan form interaktif. Aplikasi ini digunakan untuk melakukan proses pendaftaran secara online dengan validasi input sederhana.

Website ini telah di-deploy dan dapat diakses melalui:
👉 [https://formulir-pendaftaran-imphnen.netlify.app/](https://formulir-pendaftaran-imphnen.netlify.app/)

Repository GitHub:
👉 [https://github.com/viesloxy/formulir-pendaftaran-IMPHNEN.git](https://github.com/viesloxy/formulir-pendaftaran-IMPHNEN.git)

---

## 🎯 Tujuan

* Menerapkan konsep dasar JavaScript
* Memahami penggunaan event pada form
* Melakukan validasi input user
* Membuat tampilan UI yang sederhana dan user-friendly

---

## 🛠️ Teknologi yang Digunakan

* HTML
* CSS
* JavaScript (Vanilla JS)

---

## ✨ Fitur Utama

* Input data pendaftaran
* Validasi form (tidak boleh kosong)
* Notifikasi jika data belum lengkap
* Tampilan sederhana dan responsif

## ⚙️ Cara Menjalankan Project

1. Clone repository

```bash
git clone https://github.com/viesloxy/formulir-pendaftaran-IMPHNEN.git
```

2. Masuk ke folder project

```bash
cd formulir-pendaftaran-IMPHNEN
```

3. Buka file `index.html` di browser
